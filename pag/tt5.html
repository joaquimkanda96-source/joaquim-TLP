O CSS

O que é CSS?

CSS significa Cascading Style Sheets. É uma linguagem usada para estilizar páginas web. Enquanto o HTML cria a estrutura da página, o CSS define a aparência visual.


Exemplos do que o CSS faz:

  • Mudar cores dos textos
  • Alterar tamanho das letras
  • Definir cores de fundo
  • Posicionar elementos na página
